# Model weights

The original `vit_small_patch16_384.npz` file is not included in this clean
GitHub upload package because it is approximately 88.9 MB and cannot be
uploaded through GitHub's web interface.

The training code can be published and inspected without this file. If the
weight is needed later, publish it as a GitHub Release asset or upload it with
Git LFS, then add its download URL and checksum to this document.
